# Система детеции падения человека на видео

Десктопное демонстрационное приложение на базе PyQt6 и OpenCV для визуализации результатов работы ИИ-модели (YOLOv8) по фиксации фактов падения объектов/людей на видео.Приложение предназначено для сравнения исходного видеопотока и обработанного результата с выводом динамической телеметрии, расчетом метрик валидации (mAP, Precision, Recall) и графиком уверенности в реальном времени.

## Основной функционал

В реальном времени рассчитываются и обновляются параметры: mAP, Precision, Recall, Средняя уверенность модели

Интерактивный график: Построение live-кривой изменения уверенности детектора с использованием высокопроизводительной библиотеки pyqtgraph.

## Инсталляция и запуск

1. Подготовка окружения

Убедитесь, что у вас установлен Python версии 3.10 или выше.

2. Установка зависимостей

Установите необходимый набор библиотек через pip:

```bash
pip install PyQt6 opencv-python numpy pyqtgraph ultralytics
```

3. Запуск приложения

Запустите интерфейс демонстрационного модуля: `python app.py`
